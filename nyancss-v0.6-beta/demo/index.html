Getting Started

Nyan.css comes in two different forms. You can select which version you want depending on out preference and expertise. To start using Nyan.css, all you have to do is Download one of the options below.


Download

This is standart version that comes with both the minified and unminified CSS. This options requires little to not setup.

Nyan.css -version-

CDN

You can find all the version of the CDN at github.com/ryanaunur/nyan.css

<!-- Compiled and Minified CSS -->
<link rel="stylesheet" href="-linkcss-" >

<!-- Compiled and Minified Javascript -->
<script src="-linkjs-" defer ></script>


Setup

Project Structure

After downloading, extract the files into the directory where your website is located. Your directory will look samething like this.

You'll notice that there are two sets of the files. The min means that the file is "compressed" to reduce load times. These minified files are useally used in production while it is better to use the unminified files during development. 

MyWebsite/
	|--css/
	|	|--nyan.css
	|	|--nyan.min.css
	|
	|--fonts/
	|	|--avenir/
	|
	|--js/
	|	|--nyancss.js
	|	|--nyancss.min.js
	|
	|--index.html


HTML Setup

Next you just have to make sure your link files properly in your webpage. Generally it is wise to import javascript files (ex. jQuery) at the end of the body to reduce page load time. And don't forget to add ny-app at body tag. Follow the example below on how to import Nyan.css into your webpage.

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<!-- Import Nyan.css -->
	<link rel="stylesheet" type="text/css" href="css/nyan.min.css" media="screen,projection">

	<!-- Lets browser know website is optimized for mobile -->
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body ny-app>
	
	<!-- Import javascript file here -->
	<script src="js/nyancss.min.js" defer ></script>
</body>
</html>

And here we go!

Accordion

Make div parent element with class="ny-accordion", and for Child Item Element add div with class="ny-accordion-item" and don't forget to add input and label element. For content add div element with class="ny.accordion--content" inside ny-accordion-item under input+label element.

There are 2 version of accordion, single open and multiple open. Use this parent class for they are :

Accordion Class Input Type
Single Open (* default) .ny-accordion radio
Multiple Open .ny-accordion--multiple checkbox

Add checked="checked" attribut at input checkbox / radio element for default opened accordion.

Media Queries

Nyan.CSS uses media queries to serve its scalable grid, But also has a list of queries for convenience of styling your site across devices. The sizes for the queries are:

  • Desktop HD : 1200px
  • Deskto : 1000px
  • Tablet : 750px
  • Phablet : 550px
  • Mobile : 400px
Source from getskeleton.com

Responsive Utilities

These clasess are used to show and/or hide content by device via media queries.

Use one or combination of the available classes for toggling conent across viewport breakpoints.

Classes Extra small devices (Phones)
.show-xs-* Visible
.show-sm-* Hidden
.show-md-* Hidden
.show-lg-* Hidden
.hide-xs Hidden
.hide-sm Visible
.hide-md Visible
.hide-lg Visible

.show-*--* clasess come in three variations, one for each CSS display property value:

Group of Classes CSS display
.show-*--block display: block;
.show-*--inline display: inline;
.show-*--inline-block display: inline-block;
