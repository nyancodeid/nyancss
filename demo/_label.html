Labels

Add class="ny-label" to the span / paragraf tag for a labeled text.

DefaultSuccessWarningImportantInfoInverse
<span class="ny-label">Default</span>
<span class="ny-label--success">Success</span>
<span class="ny-label--warning">Warning</span>
<span class="ny-label--important">Important</span>
<span class="ny-label--info">Info</span>
<span class="ny-label--inverse">Inverse</span>
