Responsive Utilities

Show Hide

These clasess are used to show and/or hide content by device via media queries.

Use one or combination of the available classes for toggling content across viewport breakpoints.

Classes Extra small devices (Phones)
.show-xs-* Visible
.show-sm-* Hidden
.show-md-* Hidden
.show-lg-* Hidden
.hide-xs Hidden
.hide-sm Visible
.hide-md Visible
.hide-lg Visible

.show-*--* clasess come in three variations, one for each CSS display property value:

Group of Classes CSS display
.show-*--block display: block;
.show-*--inline display: inline;
.show-*--inline-block display: inline-block;
Layouting

These clasess are used to change layout content by device via media queries.

Use one or combination of the available classes for toggling layout across viewport breakpoints.

Classes Extra small devices (Phones)
ny-flex--col ny-flex-xs--row Row
ny-flex--col ny-flex-sm--row Column
ny-flex--col ny-flex-md--row Column
ny-flex--col ny-flex-lg--row Column
