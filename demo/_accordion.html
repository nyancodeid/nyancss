Accordion

There are 2 type of accordion, single open and multiple open. Use this attribute for they are :

Attribute Value Place Description
data-open single .ny-accordion Accordion Single Open
data-open multi .ny-accordion Accordion Multiple Open
data-arrow-color Hexa Color. *ex. #FFFFFF .ny-accordion Give arrow icon color.
data-component .ny-accordion-group ny-accordion Initialize ny-accordion without jquery code. wrap .ny-accordion with .ny-accordion-group

or using jquery code. 

$(document).ready(function() {
	...
	$('#foo').ny_accordion(); // default
 
	$('#bar').ny_accordion({
		singleOpen: true // true : for single open, false : for multiple open
	}); 
});
