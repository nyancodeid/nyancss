List Item

Make div parent element with class="ny-lists", and for Child Item Element add div with class="ny-list-item". For text title or header add div element with class="ny-list-item-text" inside ny-list-item.

in default List Item not present border at bottom, to show them replace class="ny-list-item" to class="ny-list-item--bordered"
