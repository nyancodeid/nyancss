Layouting

Nyancss Layout features provide sugar to enable developers to more easily create moder, responsive layout on top of CSS3 flexbox. Using Class an easy way to set a layout (eg. class="ny-flex--row").

header
sidebar
main
First item in row
Second item in row
First item in column
Second item in column
<!-- Layout 1 -->
<div class="col-xs-12 layout-1">
  <div class="ny-flex--col">
    <div class="header">header</div>
    <div class="ny-flex--row" style="height:150px;">
      <div class="sidebar col-xs-3">sidebar</div>
      <div class="main col-xs-9">main</div>
    </div>
  </div>
</div>

<!-- Layout 2 -->
<div class="col-xs-12 layout-2">
  <div class="ny-flex--row">
    <div class="ny-flex--row col-xs-6">
      <div class="col-xs-6">First item in row</div>
      <div class="col-xs-6">Second item in row</div>
    </div>
    <div class="ny-flex--col col-xs-6">
      <div>First item in column</div>
      <div>Second item in column</div>
    </div>
  </div>
</div>
