Bordered Table

Tables are bordered bottom by default.

Class Description
.ny-table--hovered Hovered effect for every table row inside tbody tag
.ny-table-cell--hovered Hovered effect for every table cell inside tbody tag
.ny-table--bordered Bordered effect for every table row and cell
