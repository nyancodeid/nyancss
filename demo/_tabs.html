Tabs

If tab has attribut data-disabled / class .disabled, then tab can't be clicked.

Attribut of Tabs

Attribut Value Places Description
data-component ny-tabs parent .ny-tabs Initialize all ny-tabs component without call it on javascript

Classes of Tabs

Class Places Description
ny-primary .ny-tabs-header Display Primary Color Theme
ny-accent .ny-tabs-header Display Accent Color Theme

jQuery callto Initialize component ny-tabs we can using data attribut or using javascript code. For more optional option visit jQuery-Tabslet - v1.7.3 on GitHub

original code by Dimitris Krestos <dkrestos@gmail.com> 
$(document).ready(function() {
	...
	$('#foo').ny_tabs(); // default
 
	$('.bar').ny_tabs({
		pagination: true, // to enable tabs pagination, default: false
		autorotate: false, // to enable auto open tabs, default: false
		active: 3 // enable tabs by default, default: 1, index from 1
		...
	}); 
});
